# AI Website Copy Generator for Local Businesses

##  Overview

This project demonstrates Prompt Engineering by designing prompts that generate complete website copy for local businesses.

The AI creates professional, SEO-friendly website content for different business types.

---

##  Features

- Hero Section
- About Us
- Services
- Why Choose Us
- Testimonials
- FAQs
- Contact Section
- SEO Meta Title
- SEO Meta Description
- Keywords
